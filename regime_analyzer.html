--

ZONAS DE SOPORTE Y RESISTENCIA
Las zonas se detectan automaticamente buscando niveles de precio donde el mercado ha reaccionado en el pasado (maximos y minimos locales repetidos). Cuanta mas "fuerza" tiene una zona, mas veces ha reaccionado el precio ahi. Estas son tus referencias para colocar ordenes programadas.
RESISTENCIAS — donde el precio puede rechazarse al alza
SOPORTES — donde el precio puede rebotar al alza
NIVEL CLAVE (LINEA CYAN EN EL GRAFICO)
Es el nivel con mas toques historicos en el periodo analizado. Generalmente el mas importante a vigilar.
--
--
COMO USAR LAS ZONAS CON TU METODO
Si el oscilador esta en amarillo (alcista) y el precio llega a una zona de soporte → es tu referencia para programar una orden de compra.

Si el oscilador esta en purpura (bajista) y el precio llega a una zona de resistencia → es tu referencia para programar una orden de venta.

Si el regimen es Range, las zonas de soporte y resistencia son los extremos del rango. El precio tiende a rebotar entre ellos.

Si el regimen es Alta Volatilidad, las zonas pueden romperse con mas facilidad. Considera ampliar el stop o reducir el tamano de la posicion.
REGIMEN ESTADISTICO DE MERCADO (HMM)
El modelo HMM (Hidden Markov Model — Modelo Oculto de Markov) analiza el comportamiento estadistico de las ultimas barras y clasifica el mercado en uno de 4 estados. No predice el precio, describe como se ha comportado el mercado recientemente y con que intensidad. La confianza indica que tan claro es ese estado segun los datos.

PROBABILIDAD DE CADA ESTADO
El modelo calcula que tan probable es estar en cada regimen en este momento. La suma siempre es 100%. El regimen con mayor probabilidad es el estado actual.
Confianza: --
QUE SIGNIFICA CADA REGIMEN
/ TENDENCIA ALCISTA
El mercado tiene presion compradora predominante. Las subidas son mas frecuentes o mas grandes que las caidas. Busca compras cuando el precio llegue a una zona de soporte.
\ TENDENCIA BAJISTA
El mercado tiene presion vendedora predominante. Las caidas son mas frecuentes o mas profundas. Busca ventas cuando el precio llegue a una zona de resistencia.
- RANGO
El precio oscila sin direccion clara. Las subidas y bajadas se compensan entre si. Opera rebotes en los extremos del rango, no sigas tendencia.
~ ALTA VOLATILIDAD
Movimientos grandes en ambas direcciones. Las zonas se rompen mas facilmente. Reduce el tamano de posicion o espera que se normalice.
COMO COMBINAR REGIMEN CON TU METODO
OSC amarillo + Regimen Alcista → contexto fuerte para buscar compras en soporte.
OSC purpura + Regimen Bajista → contexto fuerte para buscar ventas en resistencia.
OSC cualquiera + Regimen Rango → opera rebotes en extremos del rango.
Cualquier OSC + Regimen Alta Vol → reduce tamano o espera.
OSC y Regimen contradictorios → mejor no operar, espera convergencia.
